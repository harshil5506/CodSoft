## HOW TO DOWNLOAD ALL IMAGES

### Option 1: Manual Download (Recommended for verification)

1. Copy each "Direct Image URL" above
2. Paste into browser
3. Right-click → Save Image As
4. Save with the specified "File Name" in correct folder
5. Place in: `frontend/public/images/{category}/{filename}`

### Option 2: Batch Download (Windows)

Create a script or use a tool to batch download all URLs

### Image Specifications

- **Format:** JPG (JPEG)
- **Size:** Minimum 600x600px
- **File Size:** Under 200KB each (compress if needed)
- **Background:** White or clean product-only images
